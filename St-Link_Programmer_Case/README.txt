# Summary

On every Nucleo board, you can break the programmer and make a standalone ST-Link V2 with the RX/TX option. I decided to make a little enclosure to hold the programmer.

It took me 3 iterations to get the holes exactly where I wanted them.
You might need 3 drops of superglue to hold the pcb in the box.

The red and black was done with the Prusa Color3D script. There's only one layer of red in all the print.
